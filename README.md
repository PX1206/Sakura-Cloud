# SpringCloudAlibaba

项目主要架构和版本如下：  
JDK1.8  
SpringCloudAlibaba（2021.0.5.0） 分布式架构  
Nacos（2.2.0） 注册中心/配置中心  
loadbalancer 负载均衡器   
OpenFeign 远程服务调用  
Sentinel（1.8.6） 系统保护/熔断降级  
Seata（1.6.1） 分布式事务  
Gateway 网关控制  
SkyWalking（8.5.0） 服务监控/链路追踪  

想学习一下搭建过程的可以移步到https://blog.csdn.net/cl617287/article/details/131802760
